Geavanceerd: adapter & instellingen

Wordt automatisch gezet als je hierboven op "Gebruik deze instellingen" klikt. Kies cso_api handmatig voor CSO-platformsites -- dat kan de Adapter Helper niet detecteren (vereist een eigen account, zie adapters/EXAMPLES.md).



Alle adapter-specifieke configuratie (selectors, paginering, categorieën, keywords, CSO-filters) hoort hier -- zie adapters/EXAMPLES.md voor de vorm per adapter.

Minimaal voorbeeld voor html_listing (alleen item + link is genoeg; title mag weg als de link zelf de titeltekst bevat): 

{
  "start_url": "https://voorbeeld.nl/vacatures/",
  "selectors": {
    "item": "li",
    "link": "a"
  }
}


Alleen vacatures met één van deze woorden worden gemeld. Komma-gescheiden. Kan ook via settings.include_keywords -- dit veld is de eenvoudige, legacy-route en blijft als fallback werken.
