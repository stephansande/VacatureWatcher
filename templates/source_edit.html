Minimaal voorbeeld voor html_listing (alleen item + link is genoeg; title mag weg als de link zelf de titeltekst bevat): 

{
  "start_url": "https://voorbeeld.nl/vacatures/",
  "selectors": {
    "item": "li",
    "link": "a"
  }
}


Alleen vacatures met één van deze woorden worden gemeld. Komma-gescheiden. Kan ook via settings.include_keywords -- dit veld is de eenvoudige, legacy-route en blijft als fallback werken.
