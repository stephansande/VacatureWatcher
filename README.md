# VacatureWatcher


VacatureWatcher controleert automatisch vacaturepagina's van werkgevers en meldt wijzigingen.


## Functies

- Meerdere werkgevers volgen
- Eigen vacature-URL's toevoegen
- Dagelijkse of wekelijkse controle
- Nieuwe vacatures herkennen
- Verdwenen vacatures herkennen
- Zoekwoordenfilter
- E-mail meldingen
- Telegram meldingen
- Historie bewaren
- Back-ups maken
